# Обрезка ссылок с помощью ВКонтакте

Этот скрипт позволяет сокращать ссылки и получать статистику кликов по сокращённым ссылкам, используя API ВКонтакте. Он предоставляет два основных функционала: сокращение длинных ссылок в короткие с помощью сервиса ВКонтакте и получение статистики по сокращённым ссылкам, например, количество кликов по ним.

### Как установить

Для использования скрипта вам потребуется токен доступа к API ВКонтакте.

#### 1. Получение токена

1. Перейдите на страницу [создания приложения ВКонтакте](https://vk.com/dev) и создайте новое приложение.
2. Получите токен доступа для вашего приложения через [OAuth авторизацию ВКонтакте](https://vk.com/dev/access_token).
3. Сохраните ваш токен доступа.

#### 2. Установка зависимостей

Python3 должен быть уже установлен. Если его нет, скачайте и установите с [официального сайта Python](https://www.python.org/downloads/).

Затем используйте `pip` (или `pip3`, если есть конфликт с Python2) для установки зависимостей:

```
pip install -r requirements.txt
```

#### 3. Настройка токена

Для работы с API ВКонтакте вам нужно указать ваш токен доступа. Для этого создайте файл `.env` в корне проекта и добавьте в него строку:

```
VK_TOKEN=ваш_токен_доступа
```

#### 4. Установка зависимостей

Убедитесь, что все необходимые зависимости установлены. Если этого не сделано, создайте файл `requirements.txt` с содержимым:

```
requests
python-dotenv
```

Затем выполните команду:

```
pip install -r requirements.txt
```

### Как использовать

1. **Проверка, является ли ссылка сокращённой или нет**:

   Для проверки, является ли ссылка сокращённой, и получения статистики по ней, можно запустить скрипт с аргументами командной строки. Пример:

   ```
   python short_urls.py https://vk.cc/xyz123 --token YOUR_ACCESS_TOKEN
   ```

   В случае короткой ссылки будет выведено количество кликов по ней.

2. **Сокращение длинной ссылки**:

   Если вы хотите сократить длинную ссылку, передайте её в качестве аргумента, например:

   ```
   python short_urls.py https://пример123.com --token YOUR_ACCESS_TOKEN
   ```

   В ответ будет выведена сокращённая ссылка.

3. **Получение справки**:

   Для получения справки о доступных параметрах используйте флаг `--help`:

   ```
   python short_urls.py --help
   ```

   Это покажет вам описание всех аргументов и их назначения.
